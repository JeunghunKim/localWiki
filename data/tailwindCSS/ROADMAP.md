# Tailwind CSS 학습 로드맵

> Tailwind CSS를 처음 배우는 분들을 위한 단계별 학습 가이드입니다.
> 기본 개념부터 고급 기능까지 순서대로 따라가면 실무에서 활용할 수 있는 수준에 도달할 수 있습니다.

---

## Chapter 1: 소개 및 설치

- Tailwind CSS란 무엇인가?
- 기존 CSS 프레임워크와의 비교 (Bootstrap 등)
- Utility-First CSS 철학
- 설치 방법 (CDN, npm, PostCSS)
- 기본 설정 파일 (`tailwind.config.js`)
- 첫 번째 HTML 예제

## Chapter 2: 기본 유틸리티 클래스

- 색상 유틸리티 (`text-*`, `bg-*`, `border-*`)
- 크기 유틸리티 (`w-*`, `h-*`, `min-*`, `max-*`)
- 여백 유틸리티 (`m-*`, `p-*`, `space-*`)
- 표시 유틸리티 (`block`, `inline`, `hidden` 등)
- 위치 유틸리티 (`relative`, `absolute`, `fixed`, `sticky`)

## Chapter 3: 레이아웃

- Flexbox 유틸리티 (`flex`, `flex-row`, `justify-*`, `items-*` 등)
- CSS Grid 유틸리티 (`grid`, `grid-cols-*`, `col-span-*` 등)
- Container 및 box sizing
- 오버플로우 제어 (`overflow-*`)
- Z-index (`z-*`)

## Chapter 4: 반응형 디자인

- 반응형 브레이크포인트 (`sm`, `md`, `lg`, `xl`, `2xl`)
- 모바일 퍼스트 접근 방식
- 반응형 레이아웃 패턴
- 반응형 타이포그래피
- 반응형 숨김/표시

## Chapter 5: 타이포그래피 및 색상

- 폰트 패밀리, 크기, 굵기 (`font-*`, `text-*`)
- 줄 간격 및 자간 (`leading-*`, `tracking-*`)
- 텍스트 정렬 및 변환 (`text-*`, `uppercase`, `capitalize`)
- Tailwind 색상 팔레트 시스템
- 불투명도 조절 (`opacity-*`, `bg-opacity-*`)
- 다크모드 (`dark:*`)

## Chapter 6: 상태 변형 및 애니메이션

- 의사 클래스 변형 (`hover:*`, `focus:*`, `active:*`, `disabled:*`)
- 의사 요소 변형 (`before:*`, `after:*`, `placeholder:*`)
- 그룹 및 피어 변형 (`group`, `group-hover:*`, `peer`, `peer-*`)
- 트랜지션 (`transition`, `duration-*`, `ease-*`)
- 애니메이션 (`animate-spin`, `animate-ping`, `animate-bounce` 등)

## Chapter 7: 커스터마이징

- `tailwind.config.js` 상세 설정
- 테마 확장 (`theme.extend`)
- 커스텀 색상, 폰트, 크기 추가
- `@apply` 디렉티브로 재사용 가능한 스타일 생성
- 플러그인 시스템 (`@tailwindcss/typography`, `@tailwindcss/forms`)
- JIT(Just-In-Time) 모드 이해
