# Tailwind CSS 학습 자료

* 본 프로젝트는 Tailwind CSS를 처음 배우는 사람들을 위한 학습 자료를 만드는 것을 목표로 합니다.
    * 학습자료는 기본 개념부터 고급 기능까지 다루도록 작성됩니다.
    * 학습 자료는 최신 버전에 맞게 작성됩니다.
    * 학습 자료는 이해하기 쉽게 작성되도록 노력합니다.
* 학습 자료는 한글로 작성됩니다.
* 본 프로젝트는 다음의 순서로 진행됩니다:
    * ROADMAP.md 파일 생성
    * ROADMAP.md 파일을 따라 chapter별로 학습 자료 작성 (chapter#.md, chapter#.tex)
        * chapter#.md와 chapter#.tex 파일은 동일한 내용을 포함하도록 작성됩니다.
        * 각 chapter별로 실습 예제와 함께 설명이 포함되도록 작성됩니다
    * main.tex 파일을 통해 chapter#.tex 파일들을 하나로 합치기
        * 각 chapter에 제목과 섹션 번호 추가
        * TOC를 chapter별로 구분하여 작성
        * 모든 *.tex 파일은 xetex로 컴파일할 수 있도록 작성됩니다.
